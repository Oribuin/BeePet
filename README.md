# BeePet
This plugin was my first attempt at using NMS explicitly inside a plugin, I had used packets in my [Skyblock](https://github.com/Oribuin/Skyblock) for the world border but not
for much outside of that.

This took many hours of work and a lot of banging my head against walls but I eventually got it in a functional state.

I wouldn't recommend using this plugin on a production server but I believe it can be used.
